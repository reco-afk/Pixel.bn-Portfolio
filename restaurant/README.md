# Rumbia restaurant concept

Rumbia is a fictional Bruneian and Southeast Asian restaurant website created as a Pixel.bn portfolio project.

## Project type

Static multi-page front-end concept for GitHub Pages.

## Design direction

Dark forest-green editorial presentation, warm food photography, ivory display typography, restrained amber interaction color, asymmetrical imagery, and quick blur-to-focus page entrances.

## Pages

- `index.html`: image-led home page
- `menu.html`: category-based menu browser and dish details
- `reservations.html`: validated mock reservation flow
- `about.html`: concept story, ingredients, hours and fictional location

## Technology

Semantic HTML, native CSS and dependency-free JavaScript. No backend or build process is required.

## Deployment

The project is designed to be served from `/restaurant/` inside the GitHub Pages repository. Publishing must be approved by the user first.
